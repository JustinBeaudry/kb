**Reframing: vault is an index, not a content store.**

Content resolved lazily when a downstream skill needs it:
- Conversation → read transcript JSONL from path
- Code diff / state → `entire explain --checkpoint X`
- Summary → derived on-read, cached as view in `sessions/summaries/`

**What this unlocks (the interesting part):**

1. **Sessions become code-linkable.** `files_changed` in manifest (from Entire diff) = cheap index. Query "sessions that touched `worker.ts`" becomes a grep over manifests, not a search over summary text.

2. **Verifiable extraction.** `extract` can cross-reference claims against the checkpoint diff. "I introduced pattern X" → check if checkpoint actually shows X. Hallucinated summaries become detectable.

3. **Free re-summarization.** Change the prompt, swap haiku for sonnet, add new extraction heuristic — re-run over manifests, source is intact. Summaries stop being golden snapshots.

4. **Vault is KB, not MB.** Manifests are ~500 bytes. 1000 sessions = 500KB. Ships with the repo if you want.

5. **`query` skill can JOIN across transcripts + code.** "When did I last discuss auth middleware?" → find sessions whose checkpoint touched `auth/*` AND whose transcript mentions the term.

**Risks:**
- Transcript file deleted → manifest orphaned. Mitigate: keep a short conversation excerpt (first+last 2K chars) in manifest as fallback.
- Entire checkpoint lost → same. Mitigate: same excerpt covers the "what happened" half; code half degrades gracefully.

**Tradeoff:** mild fragility vs. durable architectural win + compounding queries.

*Fix: content-similarity threshold or manifest-hash-as-key (since manifest is immutable post-capture).*

**7. SessionStart inject quality regresses.** *(product-lens, conf 0.78)*
`hooks/inject` is the highest-frequency surface (fires every session start). Today it reads full summaries. Post-refactor it reads manifests (1 KB head + 1 KB tail excerpt) until `cairn summarize --all` runs. Users lose narrative context at session start.
*Fix: either inject cached summary if present, falling back to manifest excerpt; or add opportunistic catch-up summarization into SessionStart hook (reopens the deferred "no background" non-goal).*
